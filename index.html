Create or Import a Quiz

To create or import a test using the Pro-Quiz App import algorithm, type or paste a properly formatted quiz in the quiz console (if the algorithm is sucessful in adding your quiz, it will return a success message). To create a quiz, use property flags to assign properties to each quiz. Property flags and their usage are listed below:

g

Example quiz:

!(ID)testId&(ID)
!(T)Title&(T)
!(Q)What is the question number?&(Q)
!(A)2&(A, W)
!(A)6&(A, W)
!(A)1&(A, R)
!(A)3&(A, W)
